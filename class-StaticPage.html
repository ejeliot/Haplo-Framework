Class StaticPage

StaticPage - this action class finds and loads templates based on the URL entered. This would allow you to create a simple "static" site. For example: /news/sport/ would result in ../templates/news/sport.php being loaded Prefix include templates with an underscore (_) to prevent them being surfaced as actual pages
